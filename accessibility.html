Our Commitment

The GoodLife For All Foundation is committed to ensuring our website is accessible to as many people as possible, including people with visual, auditory, motor, or cognitive disabilities. We continually work to improve the accessibility and usability of this site for everyone.

Accessibility Features

This website includes the following accessibility features, available via the accessibility icon in the bottom corner of every page:

  • Larger Text — increase text size across the site for easier reading.
  • High Contrast Mode — boosts contrast between text and backgrounds.
  • Dark Mode — a lower-glare alternative colour theme, toggled from the main navigation.
  • Keyboard Navigation — the site can be navigated using a keyboard, with visible focus indicators.
  • Descriptive Alt Text — meaningful images include descriptive alternative text for screen readers.
  • Reduced Motion Support — animations are automatically minimized if your device is set to reduce motion.

Standards We Aim For

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.1, Level AA, published by the World Wide Web Consortium (W3C). These guidelines explain how to make web content more accessible for people with disabilities.

Known Limitations

Despite our best efforts, some parts of this website may not yet be fully accessible. This can include older content, third-party embedded tools, or newly added pages that have not yet been fully reviewed. We are actively working to identify and fix these issues.

Feedback

We welcome your feedback on the accessibility of this website. If you encounter any barriers or have suggestions for improvement, please let us know — your input helps us make this site better for everyone.

Contact Us

Please reach out to goodlifeforallfoundation@gmail.com or via our Contact page to report an accessibility issue. We aim to respond within a reasonable timeframe.
